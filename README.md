# my-app

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

For detailed explanation on how things work, consult the [docs for vue-loader](http://vuejs.github.io/vue-loader).

## Projectomschrijving

De opdracht is werkend op 1 probleem, het invoegen van de counter naast de provincie werkt soms niet bij allemaal omdat het te veel werk is om alles zo snel op te halen.
En het is me niet gelukt om de huidige locatie op te halen en deze automatisch active te maken.
